# Roulette

Simple roulette gaming board.  Runs best in Chrome.  Hit the `spacebar` to see the wheel spin.

## Dependencies

* [Raphael] (http://raphaeljs.com/)
* [JQuery 1.6.1] (http://jquery.com/)
